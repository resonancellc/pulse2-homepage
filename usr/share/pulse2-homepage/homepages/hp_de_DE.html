Mandriva Pulse2 Server - Startseite!

Pulse2 Agents

Agent Pack herunterladen und auf Ihr Computer installieren. Das Päckchen unterstützt Microsoft Windows bis zur Version 7 als 32- sowie 64 Bits Variante.

Weitere Betriebssysteme werden in der Dokumentation beschrieben.
