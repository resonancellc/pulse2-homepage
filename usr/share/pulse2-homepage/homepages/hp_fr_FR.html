Agent Pulse2

Téléchargez et installez l'agent Pulse2 sur votre ordinateur. Ce logiciel est compatible Windows toutes versions sur architectures 32 et 64 bits.

Pour les autres systèmes d'exploitation, se référer à la documentation.
